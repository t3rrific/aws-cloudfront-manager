Composer-based TYPO3 Installations

Execute the following command on the command line to install the missing extension if your TYPO3 installation uses Composer:

composer require aws/aws-sdk-php

Legacy TYPO3 Installations

To install the missing extension in a legacy TYPO3 installation (without Composer), follow these steps:

  • Make sure you're logged in at the TYPO3 backend as an administrator user.
  • Open the Extension Manager and update the extension list if required.
  • Search for the extension key aws_sdk_php.
  • Click the install icon next to the extension.
